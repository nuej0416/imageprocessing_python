## ✨OVERVIEW
#### Python numpy와 openCV를 이용하여 이미지를 편집하고 이미지 처리를 할 수 있는 프로그램 개발
## ⚙TECHNOLOGY
#### Python 3.8, MySQL , OpenCV 4.4.0.44, tensorflow 2.3.1, numpy 1.19.4
## 🖼RESULT
- DB에 이미지 저장, 회원 정보 저장
- Numpy, openCV를 이용한 이미지 처리
- Harrcascade 모델을 이용한 얼굴인식 이미지 처리
